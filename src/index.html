An example of CSS Grid where the elements are spanning across a defined area

In this grid structure the content has been made to span across specific grid columns. For example the header element spans across the entire grid (This is a 5 column grid). The extent of the span is defined in the css grid layout itself. This gives a substantially large amount of control with regards to the placement of the elements. On this page the sidebar on the right or this content area span across 3 grid columns and 2 grid columns respectively, in order to insert a blank space between them or if you want to change the span of the elements then just insert a "." in the grid-template-areas property.
